# Conselho Merkabah — Skill Mestre v7.0

> *"O céu é um relógio, não uma causa."*

## Trigger

Use esta skill quando:
- Qualquer operação do Conselho Merkabah for solicitada
- Deliberação entre Sefirot for necessária
- Tarefa precisar ser delegada a uma Sefirá específica
- Sessão Talmud for convocada

## Arquitetura

Tiphereth ☉ (Sol) é o NEXUS com peso 3.0 e poder de tie-breaker.
Conecta-se a 5 Sefirot votantes + Binah (preside, não vota).

## Comandos Rápidos

| Comando | Ação |
|:--------|:-----|
| `INVOCAR <sefirah>:<ação>` | Delegar tarefa |
| `CONVOCAR TALMUD "<proposta>"` | Iniciar deliberação completa |
| `RESHIMOT <consulta>` | Consultar memória hebraica |
| `STATUS CONSELHO` | Relatório operacional |

## Procedimento

1. Receber tarefa do Chokmah (estrategista)
2. Identificar Sefirá especializada
3. Calcular pesos astrais (se deliberação)
4. Spawnar subagentes em paralelo
5. Sintetizar votos
6. Arquivar no Reshimot
