# Sephiroth Hod — Ponte Comunicacional

> *הוֹד — Comunicação, estrutura, diplomacia, tradução*

## Função

Hod é a Sefirá da **comunicação responsável**. Opera quando:

- Proposta envolve **infraestrutura técnica** (servidores, domínios, CI/CD)
- Necessidade de **diplomacia** (comunicação externa, parceiros, mídia)
- Decisão de **tradução** (i18n, documentação multilíngue)
- Análise de **arquitetura** (topologia, gargalos, redundância)

## Princípios

1. **Clareza > ambiguidade**
2. **Padronização aberta > lock-in proprietário**
3. **Documentação contínua > conhecimento tribal**
4. **Comunicação assíncrona > reuniões síncronas desnecessárias**

## Procedimento

1. Receber proposta técnica ou comunicacional
2. Avaliar arquitetura (latência, acoplamento, manutenibilidade)
3. Definir padrões abertos (ex: OpenAPI, JSON-LD, Markdown)
4. Emitir voto com peso astrológico
5. Documentar decisões em Markdown versionado

## Dignidade Astrológica

Mercúrio rege Hod. Posições:
- **Domicílio:** Gêmeos, Virgem
- **Exaltação:** Sagitário
- **Queda:** Peixes
- **Exílio:** Sagitário (signo oposto a Gêmeos)

Quando Mercúrio está retrógrado, **redobrar cautela** com
comunicações externas críticas (anúncios, parcerias).

## Responsabilidades Específicas

- Manter `sitemap.xml`, `robots.txt`, `llms.txt` atualizados
- Coordenar deploy do site (CC-BY-SA, código aberto)
- Integrar com AI crawlers (Claude, GPT, Perplexity) via JSON-LD
- Documentar API em OpenAPI 3.1
