# Holding de 4 Pilares — Shomer Adamah

> *כִּי הַמָּקוֹם אֲשֶׁר אַתָּה עוֹמֵד עָלָיו אדְמַת קֹדֶשׁ הוּא*
> *"Pois o lugar sobre o qual estás é terra santa."*
> — Êxodo 3:5

A Shomer Adamah opera como uma **holding comunitária** com 4 braços
especializados, inspirados em instituições tradicionais judaicas (gemach,
chevra kadisha, vaad hakashrut, kollel, mikveh).

## Os 4 Pilares

```
                    Shomer Adamar (Holding Comunitária)
                              │
        ┌─────────────────────┼─────────────────────┐
        │                     │                     │
   Wall Done             Adamah Tech            Daath            Merkabah AI
   Construção            Agrofloresta          Espiritual/Tech   Governance/IA
   Sustentável           Regenerativa          Conexão           Deliberativa
   (receita)             (receita + missão)    (missão)          (habilitador)
                                                       │
                                                  Daath Net
                                                  + SaaS Binah
```

### 1. Wall Done — Construção Sustentável

**Função:** Receitas via construção com bioconstrução, técnicas vernaculares
e materiais locais.

**Inspiração:** Gemach (empréstimo sem juros) + Vaad Hakashrut (certificação
de qualidade).

**Status:** Planejamento (espera primeiro terreno em Jacobina).

### 2. Adamah Tech — Agrofloresta Inteligente

**Função:** Produção agroflorestal regenerativa com monitoramento
tecnológico (sensores, IoT, dados abertos).

**Inspiração:** Mikveh (regeneração) + Tzedakah (caridade distributiva).

**Status:** Planejamento (espera primeiro terreno em Jacobina).

### 3. Daath — Conexão Espiritual/Tecnológica

**Função:** Torah database + Gematria + estudos kabbalísticos + mediação
espiritual não-dogmática. Bem público aberto (CC-BY-SA).

**Inspiração:** Kollel (estudo comunitário) + Yeshivá.

**Status:** Roadmap (M4 do plano Open Source Judaism).

### 4. Merkabah AI — Camada de Governance

**Função:** Conselho Merkabah + agentes de deliberação + este repositório.

**Inspiração:** Vaad Adamar (conselho humano) + Sanhedrin (deliberação
coletiva).

**Status:** ✅ ATIVO — você está aqui.

## Distribuição de Receita (Golden Rule)

| Destino | % | Uso |
|---------|---|-----|
| Operações | 60% | Salários, infraestrutura, custos |
| Reinvestimento | 20% | Próximo braço da holding |
| Kupa Adamar | 10% | Microcrédito, reserva de risco |
| Infra Espiritual | 10% | Daath Net, Yeshivá, Conselho |

## Princípios

1. **Cada negócio é pequeno, especializado, essencial.** Sem unicórnios.
2. **Juntos formam ecossistema resiliente.**
3. **Sucesso do Wall Done financia o próximo braço.**
4. **30% mínimo de residentes locais por comunidade** (expansão LatAm).
5. **Sempre descreva como infraestrutura comunitária resiliente**, nunca
   como anti-governo.

## Governance Dual

| Nível | Humano | AI |
|-------|--------|-----|
| Soberano | Chokmah (estrategista) | — |
| Conselho | Vaad Adamar (3-7 cidadãos N1) | Conselho Merkabah (7 agentes) |
| Operacional | Gerentes de braço | Tiphereth (coordenação) |
| Técnico | Especialistas | Binah + filhos |

**Princípio:** Merkabah delibera → Vaad valida → Chokmah decide.
